# Exécution du code #

# Sous Windows :
Ouvrir une invite de commande CMD et se placer à la racine du projet
Entrez la commande "demarrer.bat"

L'application devrait normalement s'être lancée


# Sous Linux :
Ouvrir un terminal et se placer dans ce répertoire (Groupe8-SAE125)
Entrer la commande "./demarrer.sh"

L'application devrait normalement s'être lancée


# Une fois l'application lancée :

Afin de faire apparaitre un graphe MPM, cliquez sur "Fichier" et "Charger fichier" (Ou "Ctrl+N" avec les touches du clavier)
Sélectionnez le fichier .data que vous souhaitez dans le dossier "data"

Vous aurez accès à votre graphe et pourrez essayer les différentes fonctionnalités de l'application



# Fonctionnalités #

-Possibilité de faire apparaitre les dates au plus tôt puis les dates au plus tard
-Possibilité de faire apparaitre le (ou les) chemin(s) critique(s)
-Possibilité de changer les dates au plus tôt/au plus tard au format JJ/MM
-Possibilité de modifier la date de début de projet (Mis de base à la date du jour)
-Possibilité d'ajouter/supprimer/modifier des tâches
-Possibilité de modifier l'emplacement des tâches sur le graphe
-Possibilité de voir les tâches précédentes et suivantes d'une tâche
-Possibilité de réinitialiser le graphe à sa position initiale
